Estudando varias tags e como elas se comportam

HTML5 é a versão mais recente da linguagem de marcação HTML*,
introduzindo novos elementos e atributos que facilitam a criação de páginas web mais dinâmicas e interativas.
Ele suporta multimídia nativa, como áudio e vídeo, sem a necessidade de plugins externos.
Além disso, oferece melhorias em acessibilidade e semântica, permitindo um melhor SEO. Com APIs integradas,
como a de armazenamento local e canvas, HTML5 revolucionou o desenvolvimento web.

Surgimento do HTML

  • Motivação inicial: No CERN ,
    Berners-Lee percebeu que havia dificuldade na troca de documentos entre diferentes computadores e sistemas.
    Ele queria criar uma solução que conectasse documentos por meio de hipertextos,
    ou seja, links que permitissem a navegação fácil entre páginas relacionadas.

  • World Wide Web:
    Em 1989, Berners-Lee propôs a ideia da "World Wide Web" (WWW),
    um sistema que permitia criar e acessar documentos interligados na internet.
    O HTML foi essencial para essa estrutura, pois permitia a formatação e a interconexão de conteúdo por meio de links.

  • Primeira versão do HTML:
    A primeira versão informal do HTML foi apresentada por Berners-Lee em 1991.
    Ela era bem simples, com apenas 18 elementos, alguns dos quais ainda são usados hoje,
    como a (link), p (parágrafo), h1 (título), e img (imagem).
    O objetivo inicial era permitir a criação de páginas estáticas com links e textos,
    algo suficiente para divulgar informações científicas.

  • Primeiro navegador e servidor:
    Junto com o HTML,
    Berners-Lee também desenvolveu o primeiro navegador, chamado WorldWideWeb (mais tarde renomeado Nexus), e o primeiro servidor web.
    Em 1993, o lançamento do navegador Mosaic popularizou o uso do HTML e a web começou a se expandir rapidamente.

Html não é linguagem de programação

Mas porque?

HTML não é uma linguagem de programação, mas sim uma Linguagem de marcação!
(Pelo seu nome: Hypertext Markup Language) O html é usado para estruturar e formatar o conteúdo em páginas web
utilizando marcadores ou tags.

Essas tags não são executadas como códigos, mas sim utilizadas para "marcar" ou "indicar" diferentes partes de um
documento, como textos, links, imagens, cabeçalhos, listas, tabelas, entre outros.
